# 动物识别专家系统

#### 介绍
动物识别专家系统，使用Python flask框架作为后天服务器，vue作为前台web展示结果，更改自GitHub上的项目，进行了一些改进和优化，人工智能导论课程作业

#### 软件架构
服务器端 <-> 客户端


#### 安装教程
- 运行flask.py文件
- 运行 命令 ` npm run server`
